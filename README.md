# Reciepty

## Overview
Reciepty is a local web application designed for managing and tracking expenses. It operates locally on your machine without requiring an internet connection or pre-installed software such as Python.

## Getting Started

1. Download and extract the application directory.
2. Navigate to the extracted directory and double-click the `Reciepty.exe` file.
3. A terminal window will open in the background to host the local server. Your default web browser will automatically open and navigate to the application interface.

## Usage

- **Add Receipts:** Enter receipt details manually or import them using a supported URL.
- **Organization:** Categorize your receipts by assigning tags and marking important items.
- **Analysis:** Filter your expenses by month or specific tags, and view your total spending.

## Shutting Down

To safely stop the application, close the browser tab containing the application interface and close the background terminal window. All data is saved automatically in a local database.
